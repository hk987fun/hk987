0×1.windows启用超级管理员账号要点

在cmd下测试下面的命令，视频中用到的命令：

					
					//显示系统当前有哪些用户
					net user

					//激活超级管理员账户
					net user administrator /active:yes

					//赋予超级管理员账户密码123456，这个命令可以用来更改其他用户密码，语法是 net user Username Password
					net user administrator 123456

					//注销当前用户
					logoff

接下来就能在登录界面看到Administrator用户，用密码123456登录即可。
